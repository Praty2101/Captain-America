# Employee Management System (Spring IoC)

This project demonstrates Spring Core concepts:
- Inversion of Control (IoC)
- Dependency Injection
- Constructor Injection
- Setter Injection
- XML Bean Configuration

## How to Run

1. Import project into Eclipse as a Java project.
2. Add Spring Core libraries to the build path.
3. Run `TestApp.java`.

## Output
Displays employee and address details using Spring IoC container.
